Lineup Dictionary

CLICK HERE TO DOWNLOAD LINEUP DICTIONARY

Lineup dictionary crispens up your gameplay by allowing you to quickly look up lineups in the middle of a game. The interface is designed to sit on top of your game and is completely navigable via hotkeys, which are completely customizable and can be used to lookup lineups in a matter of seconds.

Patch Notes(7/12/2023):
Warning! Only use lineup dictionary when there is wifi. Otherwise it will just crash. This is going to get fixed in the future. Added a system so that Lineup Dictionary downloads lineups from the StandardLineups git repository, whenever it detects that the lineups on the user's system are behind the most recent version. This allows for a smaller download file size, and it let's us update/add lineups to the program without the user having to manually search for the next version to download.
Added a loading screen
Added a feature that displays the number of lineups in each directory
Made it so that while viewing a specific lineup, it takes 18 seconds for the application to minimize, while the minimization time for browsing the lineups/directories is 6 seconds
Reduced the quality of lineup images by making them 5-bit colors rather than 8-bit colors, which helps to save storage space and reduce lineup download times.

Added more Sova lineups to bring the total up to 50 sova recon lineups.
Added a couple more Brimstone lineups.
Fixed a bug where the dark mode didn't apply to a couple of the views
